## HOW TO START PROJECT DEVELOPMENT

``` bash

#1. Install dependencies:
npm install

#2. Install json-server:
npm install -g json-server

#3. Run json-server on db.json file:
json-server db.json -w

#4. Run webpak-dev-server:
npm start
