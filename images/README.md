## ВНИМАНИЕ! Как начать свой процесс разработки:

``` bash

1. Сделать fork репозитория ArchieMollaev/RPP. В дальнейшем все действия производим у себя в форке!

2. Сделать git clone в пустую папку, где планируете работать над проектом

3. Сделать git checkout -b [имя ветки по названию таска]

	# Таким образом ответлямся от мастера с копией всех файлов и истории коммитов. 
	Спокойно работаем. В основном придется создавать свои js файлы и рабоать в них + работа с html. 
	В уже сделанные модули не влазим, иначе можем навредить. Да и необходимости нет. 
	Единственное исключение app.js. В нем подключаем свои контроллеры.

4. Пуш командой git push --set-upstream origin [название ветки] 
	
	# В репозитории создасться ваша ветка с изменинями

5. Слияние

	# После того как работа над таском завершена. Все проверено, и все работает. 
	Делайте pull request вашей ветки в ветку dev(test). Тестируйте. 
	Если багов нет,то делаете pull request из dev(test) вашего форка в dev(test) основного репозитроия у ArchieMollaev. 
	Сообщаете мне. И я делаю pull request в master основного репозитория

6. Приступаем к работе над следющим заданием

```
### КАК РАЗВЕРНУТЬ ПРОЕКТ

``` bash

#1. Install dependencies:
npm install

#2. Install json-server:
npm install -g json-server

#3. Run json-server on db.json file:
json-server db.json -w

#4. Run webpak-dev-server:
npm start

```
